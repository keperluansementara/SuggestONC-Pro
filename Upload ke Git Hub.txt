Workflow ke depannya (kalau kamu update aplikasi)

Kalau kamu mengubah kode React, cukup lakukan ini:

git add .
git commit -m "update Fitur"
git pull origin main --rebase
git push origin main


-- tambahan running deploy develop di github & production di vercel:
Development:
git checkout dev
npm run dev

Release production:
git checkout main
git merge dev
git push

-- lalu deploy ulang:
npm run build
npm run deploy

Website GitHub Pages akan otomatis update.
